# PracticeShadowing

個人利用目的でシャドーイングの練習を助けるウェブアプリケーション。

## 使用しているWEB API
- [Words API](https://rapidapi.com/dpventures/api/wordsapi)
上記のウェブサイトで登録をして/src/environments下の各xRapidApiKeyを書き直す必要がある。
これをやらないと単語の内容が見れない。
